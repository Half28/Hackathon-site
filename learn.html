Amino acid coding

In order to create proteins, a necessary building block of life, we need to make amino acids. They join together to form polypeptides, which are needed to create proteins. Amino acids are formed by tRNA, a type of RNA. RNA is like DNA's messenger, and it is special because it can travel outside of the nucleus.
